    该项目基于野火STM32F103VE开发板，使用Keil uVision5进行开发。
    使用模块：
        1.AD8232心电传感器，引脚 OUTPUT -> PC1 , 3.3V -> 3.3V , GND -> GND
        2.JFC103集成模块，引脚 VCC -> 3.3V , GND -> GND , RXD -> PA2（TX） , TXD -> PA3（RX）
        3.MLX90614红外温度传感器，引脚 VIN -> 3.3V , GND -> GND , SDA -> PB7 , SCL -> PB6

逻辑解析：
    首先，在上电之前，按照上述模块引脚进行连接，确保模块正常工作。插入32GB容量SD卡，打开电源开关后，按下reset按钮，即可启动系统。
    系统启动后，首页是四个模块选择界面：综合仪表、实时波形、数据回顾、系统设置。
    点击综合仪表，即可进入综合仪表界面，显示当前系统的运行状态，包括心率、温度、血氧和血压等。
    点击实时波形，即可进入实时波形界面，可以显示当前心电图和实时血氧波形。
    点击数据回顾，即可进入数据回顾界面，查看之前系统运行的数据。
    点击系统设置，即可进入系统设置界面，可以手动调节系统报警的生理参数阈值等。
    各子菜单右上角有返回按钮，点击即可返回上一级菜单。
    
    在综合仪表界面时，会将采集到的数据，周期性存入SD卡中。退出综合仪表界面后，不再进行数据采集。
    在数据回顾界面时，会从SD卡中读取之前采集到的数据，并绘制波形，右侧为数据平均值以及最大值、最小值等。

    注：
        各模块采样需要一段时间稳定后，才能正常显示数据。对于体温模块，他是通过捕捉物体发出的红外辐射能进行测距，正常测温时，传感器和手之间有 1-3cm 距离，它捕捉空气中的红外线。“按上去”之后，当你把手指直接按在传感器的金属外壳上时，发生了热传导。你的体温直接通过金属壳传给了内部的补偿热敏电阻。内部 DSP 算法由于基准值（环境温）被瞬间扰乱，计算出的结果会产生巨大的正向偏差，直接跳到 50°C 甚至更高。
